## Технологии

- Nextjs как фреймворк реакта
- Zustand для стейт менеджмента и сохранения локально
- NextUI для ui


## Запуск локально

Клонируйте проект

```bash
  git clone https://link-to-project
```

Перейдите в директорию проекта

```bash
  cd my-project
```

Установка зависимостей

```bash
  npm install
```

Запуск сервера

```bash
  npm run start
```
